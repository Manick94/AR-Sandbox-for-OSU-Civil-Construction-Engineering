This document is a reference for the various calls needed to use SUMO traffic simulator 
and its accompanying scripts. 

The document is sorted by the SUMO script or program that the calls relate to.

If you add to this (and please do) just make sure you leave a short comment about what 
the call does and any parameters it takes.

------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------ 



__________________________________________________________________________________________
netconvert
__________________________________________________________________________________________

The following call imports the road network stored in "berlin.osm.xml" and stores the 
SUMO-network generated from this data into "berlin.net.xml". This is for if you have 
Open Street Map files that were downloaded elsewhere :

netconvert --osm-files --geometry.remove --ramps.guess --junctions.join --tls.guess-signals --tls.discard-simple --tls.joinberlin.osm.xml -o berlin.net.xml
